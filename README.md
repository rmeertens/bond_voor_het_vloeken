# Bond voor het Vloeken

Een ludieke, educatieve eenpagina-website over de kunst van het vloeken — met
wetenschappelijke achtergrond, geschiedenis, bondsregels en kant-en-klaar
materiaal voor een spreekbeurt (groep 7/8).

## Bestanden
- `index.html` — de website
- `style.css` — vormgeving

## Lokaal bekijken
Open `index.html` in een browser. Geen build-stap nodig.
